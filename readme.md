##Teste Homework-PHP

##Passo a Passo

1° Baixe o projeto em sua máquina.

2° Crie uma base de dados chamada 'homework' <br />
Obs: SGBD (PostgresSQL)

3° Em seu terminal na raíz do projeto rode o comando: 

php artisan migrate

4° Abra o projeto em seu navegador !
